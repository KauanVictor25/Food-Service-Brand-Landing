> **Paraíso do Sabor** é uma plataforma de catálogo digital e institucional desenvolvida para a marca de doces artesanais fundada por Fabrícia Lara. O projeto oferece uma vitrine interativa para produtos como pipocas gourmet, algodão-doce e combos festivos, facilitando a conversão de vendas via contato direto pelo WhatsApp.

---

## 🚀 Funcionalidades Chave

* **🔍 Sistema de Filtragem Dinâmica:** Permite que o usuário filtre produtos por categoria (Pipoca, Tradicional, Combos) e defina um teto de preço máximo.
* **📱 Interface Responsiva:** Layout adaptável para dispositivos móveis, incluindo um menu lateral funcional acionado por botão hamburger.
* **🎠 Carrossel de Destaques:** Utilização da biblioteca Swiper para exibir imagens do catálogo de forma deslizante e interativa.
* **📄 Catálogo Externo:** Uma página dedicada para visualização do catálogo completo integrado via container de documentos.
* **💬 Checkout via WhatsApp:** Cada produto possui um link direto que abre uma conversa no WhatsApp com uma mensagem personalizada sobre o item escolhido.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Utilizado para a estruturação semântica do conteúdo e SEO.
* **CSS3 & Flexbox**: Empregado para criar o design visual, layouts flexíveis e animações de hover.
* **JavaScript (Vanilla)**: Responsável pela lógica de filtragem de dados e controle dos elementos interativos do menu e interface.
* **Swiper.js v11**: Biblioteca de terceiros utilizada para a implementação do carrossel de imagens.
* **Bootstrap Icons**: Fonte de ícones utilizada para representar as redes sociais e controles do menu.
* **Google Fonts (Roboto)**: Tipografia padrão implementada para garantir legibilidade e estética profissional.
